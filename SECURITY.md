## Security Policy

### Coverage

Please report security vulnerabilities shipped with shaughvOS images or created by our scripts (in **/boot/shaughvos/**), like insecure system configurations or insecure default software implementations.

We are **not** responsible for a security vulnerability in any package or binary shipped with Debian, Raspbian, any 3rd party repository or within the software executables themselves. In such cases please report issues to the related repository bug tracker or upstream developers.

### Reporting a Vulnerability

If you have discovered a potential security issue within shaughvOS images, scripts, configs, or software implementations, please open a private security advisory at https://github.com/RealEmmettS/shaughvOS/security/advisories, so that we may be able to investigate and fix it internally without having users' systems exposed.

Only open a public issue at https://github.com/RealEmmettS/shaughvOS/issues if you are sure that the contained information cannot be used to compromise any users' systems.
